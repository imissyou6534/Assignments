# 作业4 - 问题1 (中文)

## 问题陈述
一个理想MOS结构在T = 300 K下维持，t_ox = 0.1 μm，p-Si衬底掺杂浓度为N_A = 10^15 cm^-3。确定：
1. 费米电势φ_F
2. 当半导体表面电势φ_s = 2φ_F时的W_dep
3. 当半导体表面电势φ_s = 2φ_F时的E_s（表面电场）
4. 阈值电压V_T

## 解答

### 已知数据
- T = 300 K
- t_ox = 0.1 μm = 10^-5 cm
- N_A = 10^15 cm^-3
- n_i = 1.5×10^10 cm^-3
- ε_ox = 3.9ε_0 = 3.45×10^-13 F/cm
- ε_s = 11.7ε_0 = 1.04×10^-12 F/cm

### 第1部分：费米电势 φ_F

费米电势为：
```
φ_F = (kT/q) × ln(N_A/n_i)
φ_F = 0.0259 × ln(10^15 / 1.5×10^10)
φ_F = 0.0259 × ln(6.67×10^4)
φ_F = 0.0259 × 11.11
φ_F = 0.288 V ≈ 0.29 V
```

### 第2部分：φ_s = 2φ_F时的耗尽宽度W_dep

在强反型（φ_s = 2φ_F）时，耗尽宽度达到最大值：
```
W_dep,max = sqrt(4ε_s × φ_F / (qN_A))
W_dep,max = sqrt(4 × 1.04×10^-12 × 0.288 / (1.6×10^-19 × 10^15))
W_dep,max = sqrt(7.5×10^-9)
W_dep,max = 8.66×10^-5 cm = 0.866 μm
```

### 第3部分：φ_s = 2φ_F时的表面电场E_s

表面电场为：
```
E_s = -sqrt(4qN_A × φ_F / ε_s)
E_s = -sqrt(4 × 1.6×10^-19 × 10^15 × 0.288 / 1.04×10^-12)
E_s = -sqrt(1.77×10^8)
E_s = -1.33×10^4 V/cm
```

大小为 |E_s| = 1.33×10^4 V/cm。

### 第4部分：阈值电压V_T

对于理想MOS结构（假设n+多晶硅栅或具有适当功函数的金属栅）：

```
V_T = V_FB + 2φ_F + (sqrt(4qε_s × N_A × φ_F)) / C_ox
```

对于p衬底上的n+多晶硅栅的理想MOS：
```
V_FB ≈ 0（理想情况）
```

单位面积氧化层电容：
```
C_ox = ε_ox / t_ox = 3.45×10^-13 / 10^-5 = 3.45×10^-8 F/cm²
```

最大耗尽电荷：
```
Q_dep,max = -qN_A × W_dep,max = -1.6×10^-19 × 10^15 × 8.66×10^-5
Q_dep,max = -1.39×10^-8 C/cm²
```

阈值电压：
```
V_T = 2φ_F + |Q_dep,max| / C_ox
V_T = 2 × 0.288 + 1.39×10^-8 / 3.45×10^-8
V_T = 0.576 + 0.403
V_T = 0.979 V ≈ 0.98 V
```

## 总结

1. φ_F = 0.29 V
2. W_dep,max = 0.866 μm（在φ_s = 2φ_F时）
3. |E_s| = 1.33×10^4 V/cm（在φ_s = 2φ_F时）
4. V_T = 0.98 V
