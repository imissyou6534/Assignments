# 作业3 - 问题3 (中文)

## 问题陈述
在正向有源模式下偏置的PNP BJT内部的电子和空穴电流如下图所示。所有电流都参考I_1，即注入基区的空穴电流。确定：
1. γ
2. α_T
3. β_DC
4. I_B
5. 对于给定的晶体管，耗尽区产生的R-G电流是否如理想晶体管分析中假设的那样可忽略？解释你的答案。

## 解答

### 分析（无特定图形）

对于正向有源模式的PNP BJT，假设标准图中的典型电流关系：
- I_1 = 从发射区注入基区的空穴电流
- 一部分在基区复合损失
- 剩余空穴到达集电极

### 第1部分：发射极效率 γ

γ定义为空穴电流与总发射极电流的比值：
```
γ = I_p,注入 / I_E = I_1 / I_E
```

如果图中显示从基区到发射区的电子注入为I_e，则：
```
I_E = I_1 + I_e
γ = I_1 / (I_1 + I_e)
```

**示例：** 如果 I_e = 0.01 I_1：
```
γ = I_1 / (I_1 + 0.01I_1) = 1/1.01 = 0.99
```

### 第2部分：基区传输系数 α_T

α_T是到达集电极的空穴比例：
```
α_T = I_p,集电极 / I_p,注入 = I_C,空穴 / I_1
```

如果图显示 I_C,空穴 = 0.98 I_1：
```
α_T = 0.98
```

### 第3部分：直流电流增益 β_DC

```
α_DC = γ × α_T
```

对于示例值：
```
α_DC = 0.99 × 0.98 = 0.97
β_DC = α_DC / (1 - α_DC) = 0.97 / 0.03 = 32.3
```

### 第4部分：基极电流 I_B

基极电流包括：
1. 注入发射区的电子：I_e
2. 基区中的复合电流：I_复合 = I_1 - I_C,空穴
3. 耗尽区R-G电流（如果显著）

```
I_B = I_e + (I_1 - I_C,空穴) + I_R-G
```

对于示例：
```
I_B = 0.01I_1 + (I_1 - 0.98I_1) = 0.01I_1 + 0.02I_1 = 0.03I_1
```

如果I_1给定特定值，比如 I_1 = 1 mA：
```
I_B = 0.03 × 1 mA = 30 μA
```

### 第5部分：耗尽区的R-G电流

要确定R-G电流是否可忽略，检查电流连续性：

**理想BJT中：** 所有基极电流来自：
1. 少子注入发射区
2. 准中性基区中的复合

**如果R-G显著：** 在耗尽区的电流图中会出现不连续性。

**从图中：**
- 如果电流在结处显示平滑过渡 → R-G可忽略
- 如果基极电流有突然跳变 → R-G显著

**数学检查：**
```
I_B,计算 = I_e + I_复合
I_B,测量 = (从 I_E - I_C)
```

如果 I_B,测量 ≈ I_B,计算，则R-G可忽略。

**示例：**
```
I_E = I_1 + I_e = 1.01 I_1
I_C = 0.98 I_1
I_B,测量 = 1.01 I_1 - 0.98 I_1 = 0.03 I_1
I_B,计算 = 0.01 I_1 + 0.02 I_1 = 0.03 I_1
```

由于它们匹配，R-G电流**可忽略**。

**结论：** 如果基极电流可以完全由发射极注入和基区复合解释，则耗尽区的R-G电流可忽略，验证了理想晶体管假设。

## 总结（示例值）

1. γ = 0.99
2. α_T = 0.98  
3. β_DC = 32.3
4. I_B = 0.03 I_1
5. 如果 I_B,测量 = I_B,计算，R-G电流**可忽略**

注：实际数值取决于从提供的图中准确读取电流值。
