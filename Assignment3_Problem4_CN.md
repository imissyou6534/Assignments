# 作业3 - 问题4 (中文)

## 问题陈述
PNP BJT准中性区中的ΔnE/nE0、ΔpB/pB0和ΔnC/nC0分布如下图所示。确定：
1. V_EB的极性
2. V_CB的极性
3. V_CB的大小
4. 该PNP BJT的工作模式

## 解答

### 分析方法

对于PNP BJT，我们分析归一化过剩载流子分布以确定偏置条件。

### 第1部分：V_EB的极性

**在PNP BJT中：**
- 发射区是p型，基区是n型

**正向偏置（V_EB > 0）：**
- 空穴从发射区注入基区
- 在E-B结边界处 ΔpB/pB0 > 0

**反向偏置（V_EB < 0）：**
- 空穴浓度耗尽
- 在E-B结边界处 ΔpB/pB0 < 0

**从典型图中：**
如果ΔpB/pB0在E-B边界显示正值并向基区内递减，这表明 **V_EB > 0**（正向偏置）。

**答案：** V_EB为**正**（E-B结正向偏置）

### 第2部分：V_CB的极性

**在PNP BJT中：**
- 集电区是p型，基区是n型

**正向偏置（V_CB > 0）：**
- 在C-B结边界处 ΔpB/pB0 > 0

**反向偏置（V_CB < 0）：**
- 在C-B结边界处 ΔpB/pB0 < 0 或 ≈ 0

**从典型图中：**
如果C-B边界处的ΔpB/pB0为负值或零，这表明 **V_CB < 0**（反向偏置）。

**答案：** V_CB为**负**（C-B结反向偏置）

### 第3部分：V_CB的大小

在基区侧的C-B结处：
```
ΔpB/pB0 = e^(qV_CB/kT) - 1
```

如果从图中，C-B边界处 ΔpB/pB0 ≈ -1：
```
-1 = e^(qV_CB/kT) - 1
e^(qV_CB/kT) = 0（近似）
```

这表明强反向偏置，V_CB → 大负值。

如果C-B边界处 ΔpB/pB0 = -0.5：
```
-0.5 = e^(qV_CB/kT) - 1
e^(qV_CB/kT) = 0.5
qV_CB/kT = ln(0.5) = -0.693
V_CB = -0.693 × kT/q = -0.693 × 0.0259 = -0.018 V
```

**更实际的示例：**
如果 ΔpB/pB0 = -0.9：
```
e^(qV_CB/kT) = 0.1
qV_CB/kT = ln(0.1) = -2.303
V_CB = -2.303 × 0.0259 = -0.060 V
```

**答案：** |V_CB| 取决于图中的确切值，对于反向偏置通常范围从0.02 V到几伏特。

### 第4部分：工作模式

BJT工作模式由结偏置决定：

| 模式 | E-B结 | C-B结 |
|------|-------------|-------------|
| 正向有源 | 正向 | 反向 |
| 饱和 | 正向 | 正向 |
| 反向有源 | 反向 | 正向 |
| 截止 | 反向 | 反向 |

**从我们的分析：**
- V_EB > 0（E-B正向偏置）
- V_CB < 0（C-B反向偏置）

**答案：** PNP BJT工作在**正向有源模式**。

### 附加验证

在PNP的正向有源模式中：
- 空穴从发射区（p）注入基区（n）
- 大部分空穴穿过基区到达集电极
- ΔpB/pB0在E-B边界最高，穿过基区递减
- ΔpB/pB0在C-B边界低/负（反向偏置）
- 这与典型的载流子分布模式匹配

## 总结

1. **V_EB极性：** 正（正向偏置）
2. **V_CB极性：** 负（反向偏置）
3. **|V_CB|大小：** 从C-B处的ΔpB/pB0值读取，使用 V_CB = (kT/q)×ln(1 + ΔpB/pB0)
4. **工作模式：** 正向有源模式

注：具体数值需要从提供的图中读取归一化过剩载流子浓度。
