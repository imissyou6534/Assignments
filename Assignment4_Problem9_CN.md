# 作业4 - 问题9 (中文)

## 问题陈述
对于NMOSFET，使用重掺杂n+多晶硅作为栅电极，Q_ox = q×8×10^10 cm^-2，L = 2 μm，W = 50 μm，t_ox = 5 nm，N_A = 10^15 cm^-3。
确定：
1. V_FB
2. V_T
3. 如果希望将V_T调制到0.5 V，应该添加哪种类型的掺杂剂？剂量是多少？

## 解答

### 已知数据
- n+多晶硅栅
- Q_ox = q×8×10^10 cm^-2 = 1.28×10^-8 C/cm²
- L = 2 μm，W = 50 μm
- t_ox = 5 nm = 5×10^-7 cm
- N_A = 10^15 cm^-3
- T = 300 K，n_i = 1.5×10^10 cm^-3
- ε_ox = 3.45×10^-13 F/cm
- ε_s = 1.04×10^-12 F/cm

### 初步计算

**氧化层电容：**
```
C_ox = ε_ox / t_ox = 3.45×10^-13 / (5×10^-7) = 6.9×10^-7 F/cm²
```

**费米电势：**
```
φ_F = (kT/q) × ln(N_A/n_i)
φ_F = 0.0259 × 11.11
φ_F = 0.288 V
```

### 第1部分：平带电压V_FB

对于p衬底上的n+多晶硅栅：

功函数差（理想情况）：
```
φ_ms ≈ -(φ_F + E_g/(2q))
φ_ms ≈ -(0.288 + 0.56)
φ_ms ≈ -0.848 V
```

含氧化层电荷的平带电压：
```
V_FB = φ_ms - Q_ox/C_ox
V_FB = -0.848 - 1.28×10^-8 / 6.9×10^-7
V_FB = -0.848 - 0.0186
V_FB = -0.867 V ≈ -0.87 V
```

### 第2部分：阈值电压V_T

```
V_T = V_FB + 2φ_F + Q_dep,max/C_ox
```

最大耗尽电荷：
```
Q_dep,max = -sqrt(4qε_s × N_A × φ_F)
Q_dep,max = -1.39×10^-8 C/cm²
```

阈值电压：
```
V_T = -0.867 + 2(0.288) + 1.39×10^-8 / 6.9×10^-7
V_T = -0.867 + 0.576 + 0.020
V_T = -0.271 V
```

这是一个耗尽型器件（负V_T）。

### 第3部分：调整阈值电压到0.5 V

当前V_T = -0.271 V
期望V_T = 0.5 V
ΔV_T = 0.5 - (-0.271) = 0.771 V

要增加V_T，需要通过离子注入在表面附近添加**受主掺杂剂（p型，例如硼）**。

由于注入剂量N_impl引起的阈值电压偏移：
```
ΔV_T = qN_impl / C_ox
```

对于p型注入的正ΔV_T：
```
0.771 = qN_impl / C_ox
N_impl = 0.771 × C_ox / q
N_impl = 0.771 × 6.9×10^-7 / 1.6×10^-19
N_impl = 3.32×10^12 cm^-2
```

**答案：**
- **掺杂剂类型：** p型（硼，B）
- **剂量：** 3.32×10^12 cm^-2 ≈ 3.3×10^12 cm^-2

### 验证

新阈值电压：
```
V_T,new = V_T,old + qN_impl/C_ox
V_T,new = -0.271 + 0.771
V_T,new = 0.5 V ✓
```

## 总结

1. V_FB = -0.87 V
2. V_T = -0.27 V（耗尽模式）
3. 添加**p型掺杂剂（硼）**，剂量为**3.3×10^12 cm^-2**以实现V_T = 0.5 V
