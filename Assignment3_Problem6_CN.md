# 作业3 - 问题6 (中文)

## 问题陈述
一个Si PNP BJT，每个区域均匀掺杂。各区域的过剩少子分布如下图所示（未显示耗尽区）。耗尽区的R-G电流可忽略。D_E = 30 cm²/s，D_B = 10 cm²/s，N_E = 10^18 cm^-3，A_E = 10^-5 cm²。
1. 该BJT工作在哪种模式？解释你的答案。
2. 确定N_C。
3. 确定基区中储存的少子电荷Q_B。
4. 确定I_E。
5. 如果基区宽度调制效应可忽略，确定该BJT工作在正向有源模式时的β_DC。

## 解答

### 分析（无特定图形的一般方法）

对于PNP BJT，我们分析过剩少子载流子分布：
- Δn_E 发射区中的电子
- Δp_B 基区中的空穴
- Δn_C 集电区中的电子

### 第1部分：工作模式

**从过剩载流子分布的指标：**

**正向有源模式：**
- E-B正向偏置：E-B边界处 Δp_B > 0
- C-B反向偏置：C-B边界处 Δp_B ≈ 0
- Δp_B穿过基区线性递减

**饱和模式：**
- E-B正向偏置：E-B边界处 Δp_B > 0
- C-B正向偏置：C-B边界处 Δp_B > 0
- Δp_B在整个基区保持高值

**可能答案：** 如果图显示Δp_B从E-B处的高值递减到C-B处接近零，BJT工作在**正向有源模式**。

### 第2部分：确定N_C

从集电区的电子分布Δn_C：

在C-B结边界处：
```
Δn_C(0) = n_C0 × (e^(qV_CB/kT) - 1)
```

其中 n_C0 = n_i²/N_C

如果V_CB < 0（反向偏置），Δn_C(0) → 0或负值。

从 n_C0 = n_i²/N_C：
```
N_C = n_i² / n_C0
```

**示例：** 如果 n_C0 = 10^4 cm^-3：
```
N_C = (1.5×10^10)² / 10^4 = 2.25×10^16 cm^-3
```

### 第3部分：基区中储存的少子电荷 Q_B

对于基区中Δp_B的线性分布：
```
Q_B = q × A_E × ∫[0 to W_B] Δp_B(x) dx
```

如果Δp_B从E-B处的Δp_B(0)线性变化到C-B处的0：
```
Δp_B(x) = Δp_B(0) × (1 - x/W_B)
```

则：
```
Q_B = q × A_E × Δp_B(0) × W_B / 2
```

**示例：** 如果 Δp_B(0) = 10^10 cm^-3，W_B = 1 μm = 10^-4 cm：
```
Q_B = 1.6×10^-19 × 10^-5 × 10^10 × 10^-4 / 2
Q_B = 8×10^-19 C = 0.8 pC
```

### 第4部分：确定I_E

发射极电流有两个分量：

**空穴电流（主要）：**
```
I_p = q × A_E × D_B × (dΔp_B/dx)|_{x=0}
```

对于线性分布：
```
dΔp_B/dx|_{x=0} = -Δp_B(0)/W_B
```

因此：
```
I_p = q × A_E × D_B × Δp_B(0) / W_B
```

**电子电流（次要）：**
```
I_n = q × A_E × D_E × Δn_E(0) / L_E
```

**总发射极电流：**
```
I_E = I_p + I_n
```

**示例计算：**
使用 Δp_B(0) = 10^10 cm^-3，W_B = 10^-4 cm，D_B = 10 cm²/s：
```
I_p = 1.6×10^-19 × 10^-5 × 10 × 10^10 / 10^-4
I_p = 1.6×10^-8 A = 16 μA
```

如果 I_n << I_p，则 I_E ≈ 16 μA。

### 第5部分：确定正向有源模式的β_DC

```
β_DC = I_C / I_B
```

在正向有源模式中：
```
I_C ≈ I_p（对于PNP，从发射区的空穴电流）
I_B = I_n + (I_E - I_C) = I_n + 复合电流
```

或者使用：
```
β_DC = γ × α_T / (1 - γ × α_T)
```

**发射极效率：**
```
γ = I_p / (I_p + I_n)
```

**基区传输系数（短基区）：**
```
α_T ≈ 1 - W_B²/(2D_B×τ_B)
```

**示例：** 如果 γ = 0.95，α_T = 0.99：
```
β_DC = (0.95 × 0.99) / (1 - 0.95 × 0.99)
β_DC = 0.9405 / 0.0595 = 15.8
```

## 总结

1. **工作模式：** 正向有源模式（基于Δp_B分布）
2. **N_C：** 从Δn_C和结偏置计算（示例：~10^16 cm^-3）
3. **Q_B：** q × A_E × Δp_B(0) × W_B / 2（从图中读取）
4. **I_E：** q × A_E × D_B × Δp_B(0) / W_B + 电子分量
5. **β_DC：** 从γ和α_T计算，或从电流比计算

注：具体数值需要从提供的图中读取过剩载流子分布。
