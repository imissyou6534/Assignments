# 作业3 - 问题5 (中文)

## 问题陈述
两个Si NPN BJT，#1和#2，除了W_B1 > W_B2外完全相同。此外，N_E >> N_B > N_C，并且两个晶体管中W << L_B。在相同的正向有源模式下，哪个BJT将表现出：
1. 更大的发射极效率γ？解释你的答案。
2. 更大的基区传输系数α_T？解释你的答案。
3. 更大的β_DC？解释你的答案。
4. 对基区宽度调制效应更敏感？解释你的答案。
5. 更大的穿通电压？解释你的答案。

如果假设最大输出电流受载流子倍增和雪崩限制，哪个BJT将表现出：
6. 更大的V_CB0？解释你的答案。
7. 更大的V_CE0？解释你的答案。

## 解答

### 第1部分：发射极效率 γ

发射极效率为：
```
γ = I_En / (I_En + I_Ep) = 1 / (1 + I_Ep/I_En)
```

对于NPN BJT：
```
I_Ep/I_En ∝ W_B
```

**因此：** BJT #2（W_B2较小）有**更大的γ**，因为流回发射区的空穴电流更少。

### 第2部分：基区传输系数 α_T

对于短基区（W << L_B）：
```
α_T = 1 - W_B²/(2L_n²)
```

**因此：** BJT #2（W_B2较小）有**更大的α_T**，因为：
- 较薄基区中的复合更少
- α_T ≈ 1 - W_B²/(2D_n×τ_n)
- 更小的W_B → 更大的α_T

### 第3部分：直流电流增益 β_DC

```
β_DC = (γ × α_T) / (1 - γ × α_T)
```

由于BJT #2同时具有更大的γ和α_T：

**因此：** BJT #2（W_B2较小）有**更大的β_DC**。

**解释：** 发射极效率和基区传输系数都随着更薄的基区而改善，导致更高的电流增益。

### 第4部分：对基区宽度调制的敏感性（厄利效应）

基区宽度调制导致：
```
ΔW_B / W_B
```

基区宽度的相对变化影响α_T，进而影响I_C。

对于相同的ΔW_B，相对变化ΔW_B/W_B对于较小的W_B更大。

**因此：** BJT #2（W_B2较小）对基区宽度调制**更敏感**。

**解释：** 当基区较薄时，耗尽宽度的给定变化代表总基区宽度的更大分数变化，使厄利效应更明显。

### 第5部分：穿通电压

当C-B耗尽区到达E-B耗尽区时发生穿通，即：
```
x_C,CB = W_B
```

由于W_B1 > W_B2：

**因此：** BJT #1（W_B1较大）有**更大的穿通电压**。

**解释：** 更宽的基区需要C-B结上更高的反向偏压才能使耗尽区跨越整个基区宽度。

### 第6部分：击穿电压 V_CB0

V_CB0是发射极开路时的集电极-基极击穿电压。

击穿电压主要由以下因素决定：
- 集电极掺杂N_C
- 基区掺杂N_B  
- C-B耗尽区的电场

由于两个晶体管具有相同的掺杂分布：

**因此：** **两个BJT具有相同的V_CB0**。

**解释：** 击穿电压取决于结掺杂浓度，而不取决于基区宽度。由于两个器件的N_C和N_B相同，V_CB0相同。

### 第7部分：击穿电压 V_CE0

V_CE0是基极开路时的集电极-发射极击穿电压。

关系式为：
```
V_CE0 ≈ BV_CBO / √(β_DC)
```

由于BJT #2具有更大的β_DC：

**因此：** BJT #1（由于W_B1较大而β_DC较小）有**更大的V_CE0**。

**解释：** 更高的β_DC通过晶体管作用导致载流子的更有效倍增，在较低的V_CE下达到雪崩击穿。β_DC较低的晶体管在击穿前可以承受更高的V_CE。

## 总结

| 参数 | 在BJT #1或#2中更大 | 原因 |
|------|------------------|------|
| γ | #2（较小W_B） | 向发射区的空穴注入更少 |
| α_T | #2（较小W_B） | 基区中的复合更少 |
| β_DC | #2（较小W_B） | 更高的γ和α_T |
| 厄利效应敏感性 | #2（较小W_B） | 更大的ΔW_B/W_B比 |
| V_PT | #1（较大W_B） | 需要穿通的基区更宽 |
| V_CB0 | 相同 | 相同的掺杂分布 |
| V_CE0 | #1（较大W_B） | 较低的β_DC |
