# 作业2 - 问题6 (中文)

## 问题陈述
对于室温下的Si pn长二极管。N_D = 10^18 cm^-3，N_A = 10^16 cm^-3，τ_p = τ_n = 10^-8 s，D_n = 31 cm^2/s，D_p = 12 cm^2/s，A_E = 10^-4 cm^2。确定以下偏置条件下的G_D、C_D和C_J：
1. V_A = 0.1 V
2. V_A = 0.5 V
3. V_A = 0.7 V
4. V_A = 0 V
5. V_A = -5 V

## 解答

### 已知数据
- T = 300 K，n_i = 1.5×10^10 cm^-3
- N_D = 10^18 cm^-3，N_A = 10^16 cm^-3
- τ_p = τ_n = 10^-8 s
- D_n = 31 cm^2/s，D_p = 12 cm^2/s
- A_E = 10^-4 cm^2
- ε_s = 1.04×10^-12 F/cm

### 初步计算

**扩散长度：**
```
L_n = sqrt(D_n × τ_n) = sqrt(31 × 10^-8) = 5.57×10^-4 cm = 5.57 μm
L_p = sqrt(D_p × τ_p) = sqrt(12 × 10^-8) = 3.46×10^-4 cm = 3.46 μm
```

**平衡少子浓度：**
```
p_n0 = n_i^2/N_D = (1.5×10^10)^2/10^18 = 2.25×10^2 cm^-3
n_p0 = n_i^2/N_A = (1.5×10^10)^2/10^16 = 2.25×10^4 cm^-3
```

**内建电势：**
```
V_bi = (kT/q) × ln(N_A×N_D/n_i^2)
V_bi = 0.0259 × ln(10^34/(2.25×10^20))
V_bi = 0.0259 × 31.65 = 0.82 V
```

**饱和电流I_S：**
```
I_S = qA_E(D_p×p_n0/L_p + D_n×n_p0/L_n)
I_S = 1.6×10^-19 × 10^-4 × (12×2.25×10^2/3.46×10^-4 + 31×2.25×10^4/5.57×10^-4)
I_S ≈ 2.0×10^-14 A
```

### 第1部分：V_A = 0.1 V

```
I_D = I_S(e^(qV_A/kT) - 1) = 2.0×10^-14 × (e^3.86 - 1) ≈ 9.38×10^-13 A
G_D = (q/kT) × I_S × e^(qV_A/kT) = 3.7×10^-11 S
C_D = (τ_T/kT) × I_D = 3.62×10^-19 F
C_J ≈ 5.8×10^-13 F
```

### 第2部分：V_A = 0.5 V

```
I_D ≈ 4.8×10^-6 A
G_D = 1.85×10^-4 S
C_D = 1.85×10^-12 F
C_J ≈ 8.7×10^-13 F
```

### 第3部分：V_A = 0.7 V

```
I_D ≈ 1.1×10^-2 A
G_D = 0.425 S
C_D = 4.25×10^-9 F
C_J ≈ 1.4×10^-12 F
```

### 第4部分：V_A = 0 V

```
I_D = 0 A
G_D = 7.7×10^-13 S
C_D = 0 F
C_J ≈ 5.4×10^-13 F
```

### 第5部分：V_A = -5 V

```
I_D ≈ -2.0×10^-14 A
G_D ≈ 0 S
C_D ≈ 0 F
C_J ≈ 2.0×10^-13 F
```
