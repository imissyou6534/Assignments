# Assignment 2 - Problem 6 (English)

## Problem Statement
For a Si pn long diode at RT. N_D = 10^18 cm^-3, N_A = 10^16 cm^-3, τ_p = τ_n = 10^-8 s, D_n = 31 cm^2/s, D_p = 12 cm^2/s, A_E = 10^-4 cm^2. Determine the G_D, C_D and C_J at following bias conditions:
1. V_A = 0.1 V
2. V_A = 0.5 V
3. V_A = 0.7 V
4. V_A = 0 V
5. V_A = -5 V

## Solution

### Given Data
- T = 300 K, n_i = 1.5×10^10 cm^-3
- N_D = 10^18 cm^-3, N_A = 10^16 cm^-3
- τ_p = τ_n = 10^-8 s
- D_n = 31 cm^2/s, D_p = 12 cm^2/s
- A_E = 10^-4 cm^2
- ε_s = 1.04×10^-12 F/cm

### Preliminary Calculations

**Diffusion lengths:**
```
L_n = sqrt(D_n × τ_n) = sqrt(31 × 10^-8) = 5.57×10^-4 cm = 5.57 μm
L_p = sqrt(D_p × τ_p) = sqrt(12 × 10^-8) = 3.46×10^-4 cm = 3.46 μm
```

**Equilibrium minority concentrations:**
```
p_n0 = n_i^2/N_D = (1.5×10^10)^2/10^18 = 2.25×10^2 cm^-3
n_p0 = n_i^2/N_A = (1.5×10^10)^2/10^16 = 2.25×10^4 cm^-3
```

**Built-in potential:**
```
V_bi = (kT/q) × ln(N_A×N_D/n_i^2)
V_bi = 0.0259 × ln(10^34/(2.25×10^20))
V_bi = 0.0259 × 31.65 = 0.82 V
```

**Saturation current I_S:**
```
I_S = qA_E(D_p×p_n0/L_p + D_n×n_p0/L_n)
I_S = 1.6×10^-19 × 10^-4 × (12×2.25×10^2/3.46×10^-4 + 31×2.25×10^4/5.57×10^-4)
I_S = 1.6×10^-23 × (7.79×10^6 + 1.25×10^9)
I_S ≈ 2.0×10^-14 A
```

### Part 1: V_A = 0.1 V

**Diode current:**
```
I_D = I_S(e^(qV_A/kT) - 1) = 2.0×10^-14 × (e^(0.1/0.0259) - 1)
I_D = 2.0×10^-14 × (e^3.86 - 1) = 2.0×10^-14 × 46.9 = 9.38×10^-13 A
```

**Conductance G_D:**
```
G_D = dI_D/dV_A = (q/kT) × I_S × e^(qV_A/kT)
G_D = (1/0.0259) × 2.0×10^-14 × 47.9 = 3.7×10^-11 S
```

**Diffusion capacitance C_D:**
```
C_D = (τ_T/kT) × I_D, where τ_T = τ_p = τ_n = 10^-8 s
C_D = (10^-8/0.0259) × 9.38×10^-13 = 3.62×10^-19 F
```

**Junction capacitance C_J:**
```
C_J = A_E × sqrt(qε_s/(2(V_bi - V_A)) × (N_A×N_D)/(N_A + N_D))
C_J = 10^-4 × sqrt(1.6×10^-19 × 1.04×10^-12/(2×0.72) × 10^34/(1.01×10^16))
C_J ≈ 5.8×10^-13 F
```

### Part 2: V_A = 0.5 V

```
I_D = 2.0×10^-14 × (e^19.3 - 1) ≈ 4.8×10^-6 A
G_D = (1/0.0259) × 4.8×10^-6 = 1.85×10^-4 S
C_D = (10^-8/0.0259) × 4.8×10^-6 = 1.85×10^-12 F
C_J = 10^-4 × sqrt(qε_s/(2×0.32) × 10^34/(1.01×10^16)) ≈ 8.7×10^-13 F
```

### Part 3: V_A = 0.7 V

```
I_D = 2.0×10^-14 × (e^27.0 - 1) ≈ 1.1×10^-2 A
G_D = (1/0.0259) × 1.1×10^-2 = 0.425 S
C_D = (10^-8/0.0259) × 1.1×10^-2 = 4.25×10^-9 F
C_J = 10^-4 × sqrt(qε_s/(2×0.12) × 10^34/(1.01×10^16)) ≈ 1.4×10^-12 F
```

### Part 4: V_A = 0 V

```
I_D = 0 A
G_D = (q/kT) × I_S = 7.7×10^-13 S
C_D = 0 F
C_J = 10^-4 × sqrt(qε_s/(2×0.82) × 10^34/(1.01×10^16)) ≈ 5.4×10^-13 F
```

### Part 5: V_A = -5 V

```
I_D ≈ -I_S = -2.0×10^-14 A
G_D ≈ 0 S
C_D ≈ 0 F
C_J = 10^-4 × sqrt(qε_s/(2×5.82) × 10^34/(1.01×10^16)) ≈ 2.0×10^-13 F
```
