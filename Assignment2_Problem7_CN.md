# 作业2 - 问题7 (中文)

## 问题陈述
对于室温下的Si n+p结。N_A = 10^17 cm^-3，D_n = 19 cm^2/s，A_E = 10^-5 cm^2。p-Si中空间电荷区边界与p-Si欧姆接触金属电极之间的宽度为3 μm。所有载流子在p-Si的欧姆接触金属电极处复合。假设流过该结的电流为0.5 mA，确定p-Si准中性区中储存电荷的总数。

## 解答

### 已知数据
- T = 300 K
- N_A = 10^17 cm^-3
- D_n = 19 cm^2/s
- A_E = 10^-5 cm^2
- W_B = 3 μm = 3×10^-4 cm (基区宽度)
- I = 0.5 mA = 5×10^-4 A
- n_i = 1.5×10^10 cm^-3

### 分析

这是一个短二极管问题，因为所有载流子在背接触处复合，意味着准中性区宽度是有限的（W_B = 3 μm）。

对于短二极管，少子载流子分布近似为线性而非指数型。

### 少子载流子分布

对于有限基区，边界条件为：
- 在x = 0处：Δn_p(0) = n_p0(e^(qV_A/kT) - 1)
- 在x = W_B处：Δn_p(W_B) = 0 (所有载流子在接触处复合)

分布为线性：
```
Δn_p(x) = Δn_p(0) × (1 - x/W_B)
```

### 电流方程

对于短二极管，电流密度为：
```
J_n = qD_n × Δn_p(0)/W_B
```

因此：
```
I = A_E × J_n = qD_n × A_E × Δn_p(0)/W_B
```

求解Δn_p(0)：
```
Δn_p(0) = I × W_B / (qD_n × A_E)
Δn_p(0) = 5×10^-4 × 3×10^-4 / (1.6×10^-19 × 19 × 10^-5)
Δn_p(0) = 1.5×10^-7 / (3.04×10^-23)
Δn_p(0) = 4.93×10^15 cm^-3
```

### 储存电荷计算

总储存少子载流子电荷为：
```
Q = q × A_E × ∫[0 to W_B] Δn_p(x) dx
```

对于线性分布：
```
Q = q × A_E × ∫[0 to W_B] Δn_p(0) × (1 - x/W_B) dx
Q = q × A_E × Δn_p(0) × [x - x^2/(2W_B)]|[0 to W_B]
Q = q × A_E × Δn_p(0) × W_B/2
```

代入数值：
```
Q = 1.6×10^-19 × 10^-5 × 4.93×10^15 × 3×10^-4 / 2
Q = 1.18×10^-12 C
```

或者，使用渡越时间关系：
```
Q = I × τ_t
```

短二极管的渡越时间为：
```
τ_t = W_B^2 / (2D_n) = (3×10^-4)^2 / (2 × 19)
τ_t = 2.37×10^-9 s
```

因此：
```
Q = 5×10^-4 × 2.37×10^-9 = 1.18×10^-12 C
```

### 储存电荷总数

```
N = Q/q = 1.18×10^-12 / 1.6×10^-19
N = 7.4×10^6 个电子
```

## 总结
p-Si准中性区中储存电荷的总数约为 **7.4 × 10^6 个电子** 或储存电荷为 **1.18 pC**。
