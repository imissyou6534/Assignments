# 作业3 - 问题1 (中文)

## 问题陈述
一个Si NPN BJT，N_E = 10^18 cm^-3，N_B = 10^16 cm^-3，N_C = 10^15 cm^-3，W_B = 2 μm，在室温下保持平衡条件。
1. 绘制器件的能带图，正确定位三个区域的费米能级。
2. 绘制BJT内部静电势（在发射区设V=0）、电场和电荷密度随位置的变化。
3. 计算集电极和发射极之间的净电势差V_CE。
4. 确定基区的准中性宽度。
5. 计算E-B和C-B耗尽区的最大电场强度。

## 解答

### 已知数据
- T = 300 K，n_i = 1.5×10^10 cm^-3
- N_E = 10^18 cm^-3 (发射区掺杂)
- N_B = 10^16 cm^-3 (基区掺杂)
- N_C = 10^15 cm^-3 (集电区掺杂)
- W_B = 2 μm (基区宽度)
- ε_s = 1.04×10^-12 F/cm

### 第1部分：能带图

对于平衡态的NPN BJT，费米能级E_F在整个器件中恒定。

**费米能级位置：**
- 发射区（n型）：E_F接近E_C（重掺杂n+）
- 基区（p型）：E_F接近E_V（p掺杂）
- 集电区（n型）：E_F在E_C和E_i之间（轻掺杂n）

能带图（文字表示）：
```
发射区      基区       集电区
(n+)       (p)        (n)

E_C ___      ____       ___
       \    /    \     /
        \  /      \   /
         \/qV_BE   \ /qV_BC
    E_F _ _ _ _ _ _ _ _ _ _  (恒定)
         /\        / \
        /  \      /   \
       /    \____/     \___
E_V __/                    
```

### 第2部分：静电势、电场和电荷密度

**在发射区设V=0：**

静电势 V(x):
```
V   |       ___________
    |      /           \___
VBE |_____/                 
    |                      \___
0   |___________________________VCE
    E    EB    B    BC    C
```

电场 E(x):
```
E   |    ^          ^
    |   /|\        /|\
0   |____.__________.__________
    E   EB    B   BC    C
```

电荷密度 ρ(x):
```
ρ   | +  |    -    |  +
    |____|_________|_______
    E   EB    B   BC    C
```

### 第3部分：净电势差V_CE

**E-B结内建电势：**
```
V_BE = (kT/q) × ln(N_E × N_B / n_i^2)
V_BE = 0.0259 × ln(10^34 / 2.25×10^20)
V_BE = 0.0259 × 31.43 = 0.814 V
```

**B-C结内建电势：**
```
V_BC = (kT/q) × ln(N_B × N_C / n_i^2)
V_BC = 0.0259 × ln(10^31 / 2.25×10^20)
V_BC = 0.0259 × 24.52 = 0.635 V
```

**净电势差：**
```
V_CE = V_BE + V_BC = 0.814 + 0.635 = 1.449 V ≈ 1.45 V
```

### 第4部分：准中性基区宽度

**E-B耗尽宽度：**
```
W_EB ≈ sqrt(2ε_s × V_BE / q × 1/N_B)  (因为 N_E >> N_B)
W_EB = sqrt(2 × 1.04×10^-12 × 0.814 / (1.6×10^-19 × 10^16))
W_EB ≈ 3.26×10^-5 cm = 0.326 μm
```

大部分W_EB延伸到基区（x_B,EB）。

**准中性基区宽度：**
```
W_B' = W_B - x_B,EB - x_B,BC
W_B' ≈ 2.0 - 0.30 - 0.09 ≈ 1.61 μm
```

### 第5部分：最大电场

**E-B结：**
```
E_max,EB ≈ 1.6×10^-19 × 10^16 × 3×10^-5 / 1.04×10^-12
E_max,EB ≈ 4.6×10^4 V/cm
```

**B-C结：**
```
E_max,BC ≈ 1.33×10^4 V/cm
```

## 总结
1. 能带图显示恒定的E_F，在结处能带弯曲
2. V_CE = 1.45 V
3. W_B' ≈ 1.61 μm
4. E_max,EB ≈ 4.6×10^4 V/cm，E_max,BC ≈ 1.33×10^4 V/cm
