# 作业4 - 问题10 (中文)

## 问题陈述
一个MOSFET具有以下参数：n+多晶硅作为栅电极，t_ox = 8 nm，N_A = 10^17 cm^-3，Q_ox = 5×10^10 cm^-2。
1. 该MOSFET的V_T是多少？
2. 该MOSFET是增强型还是耗尽型？
3. 需要什么类型的掺杂剂和剂量才能使V_T = 0？

## 解答

### 已知数据
- n+多晶硅栅
- t_ox = 8 nm = 8×10^-7 cm
- N_A = 10^17 cm^-3
- Q_ox = 5×10^10 cm^-2 = 8×10^-9 C/cm²
- T = 300 K，n_i = 1.5×10^10 cm^-3
- ε_ox = 3.45×10^-13 F/cm
- ε_s = 1.04×10^-12 F/cm

### 初步计算

**氧化层电容：**
```
C_ox = ε_ox / t_ox = 3.45×10^-13 / (8×10^-7) = 4.31×10^-7 F/cm²
```

**费米电势：**
```
φ_F = (kT/q) × ln(N_A/n_i)
φ_F = 0.0259 × ln(10^17 / 1.5×10^10)
φ_F = 0.0259 × 15.71
φ_F = 0.407 V
```

**平带电压：**

n+多晶硅在p-Si上的功函数差：
```
φ_ms = -(φ_F + E_g/(2q))
φ_ms = -(0.407 + 0.56)
φ_ms = -0.967 V
```

含氧化层电荷：
```
V_FB = φ_ms - Q_ox/C_ox
V_FB = -0.967 - 8×10^-9 / 4.31×10^-7
V_FB = -0.986 V
```

### 第1部分：阈值电压V_T

最大耗尽电荷：
```
Q_dep,max = -sqrt(4qε_s × N_A × φ_F)
Q_dep,max = -sqrt(4 × 1.6×10^-19 × 1.04×10^-12 × 10^17 × 0.407)
Q_dep,max = -5.21×10^-7 C/cm²
```

阈值电压：
```
V_T = V_FB + 2φ_F + |Q_dep,max|/C_ox
V_T = -0.986 + 2(0.407) + 5.21×10^-7 / 4.31×10^-7
V_T = -0.986 + 0.814 + 1.209
V_T = 1.037 V ≈ 1.04 V
```

### 第2部分：增强型还是耗尽型？

由于V_T = 1.04 V > 0：

**答案：增强型**

**解释：** 增强型NMOSFET需要正栅电压（V_GS > V_T > 0）来产生反型沟道并开启器件。当V_GS = 0时，器件关闭，这是增强型工作的特征。

### 第3部分：使V_T = 0所需的掺杂剂类型和剂量

当前V_T = 1.04 V
期望V_T = 0 V
ΔV_T = 0 - 1.04 = -1.04 V

要降低V_T，需要通过离子注入在表面附近添加**施主掺杂剂（n型，例如磷或砷）**。

阈值电压偏移：
```
ΔV_T = -qN_impl / C_ox
```

对于n型注入（负ΔV_T）：
```
-1.04 = -qN_impl / C_ox
N_impl = 1.04 × C_ox / q
N_impl = 1.04 × 4.31×10^-7 / 1.6×10^-19
N_impl = 2.80×10^12 cm^-2
```

**答案：**
- **掺杂剂类型：** n型（磷，P或砷，As）
- **剂量：** 2.8×10^12 cm^-2

### 验证

新阈值电压：
```
V_T,new = V_T,old - qN_impl/C_ox
V_T,new = 1.04 - 1.04
V_T,new = 0 V ✓
```

当V_T = 0时，器件成为**零阈值**或**天然** NMOSFET，处于增强型和耗尽型之间的边界。

## 总结

1. V_T = 1.04 V
2. **增强型**（V_T > 0）
3. 添加**n型掺杂剂（磷或砷）**，剂量为**2.8×10^12 cm^-2**以实现V_T = 0
