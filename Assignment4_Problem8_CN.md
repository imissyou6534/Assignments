# 作业4 - 问题8 (中文)

## 问题陈述
假设NMOSFET采用n+多晶硅栅电极，t_ox = 1.1 nm，N_A = 10^18 cm^-3，Q_ox = 0。确定：
1. V_T
2. 亚阈值摆幅S
3. 当W = 1 μm，L = 18 nm时的I_off，考虑到V_GS = V_T时I_D = 100W/L (nA)

## 解答

### 已知数据
- n+多晶硅栅
- t_ox = 1.1 nm = 1.1×10^-7 cm
- N_A = 10^18 cm^-3
- Q_ox = 0
- T = 300 K，n_i = 1.5×10^10 cm^-3
- ε_ox = 3.45×10^-13 F/cm
- ε_s = 1.04×10^-12 F/cm

### 初步计算

**氧化层电容：**
```
C_ox = ε_ox / t_ox = 3.45×10^-13 / (1.1×10^-7) = 3.136×10^-6 F/cm²
```

**费米电势：**
```
φ_F = (kT/q) × ln(N_A/n_i)
φ_F = 0.0259 × ln(10^18 / 1.5×10^10)
φ_F = 0.0259 × 18.02
φ_F = 0.467 V
```

### 第1部分：阈值电压V_T

对于Q_ox = 0的p衬底上的n+多晶硅栅：

```
V_T = 2φ_F + (1/C_ox) × sqrt(4qε_s × N_A × φ_F)
```

计算耗尽电荷：
```
Q_dep,max = -sqrt(4 × 1.6×10^-19 × 1.04×10^-12 × 10^18 × 0.467)
Q_dep,max = -5.57×10^-7 C/cm²
```

阈值电压：
```
V_T = 2 × 0.467 + 5.57×10^-7 / 3.136×10^-6
V_T = 0.934 + 0.178
V_T = 1.112 V ≈ 1.11 V
```

### 第2部分：亚阈值摆幅S

亚阈值摆幅为：
```
S = (kT/q) × ln(10) × (1 + C_d/C_ox)
```

其中C_d是耗尽电容：
```
W_dep,max = sqrt(4ε_s × φ_F / (qN_A))
W_dep,max = 1.10×10^-6 cm = 11 nm

C_d = ε_s / W_dep,max = 9.45×10^-7 F/cm²
```

亚阈值摆幅：
```
S = 0.0259 × 2.303 × (1 + 9.45×10^-7 / 3.136×10^-6)
S = 0.0596 × 1.301
S = 77.5 mV/decade ≈ 78 mV/decade
```

### 第3部分：I_off

给定：V_GS = V_T时I_D = 100W/L (nA)

W = 1 μm，L = 18 nm：
```
I_on = 100 × (1×10^-4 / 18×10^-7) nA
I_on = 5.556 μA
```

关态电流I_off发生在V_GS = 0 V时。

使用亚阈值电流：
```
I_D = I_0 × e^(qV_GS/(nkT))
```

其中n = 1 + C_d/C_ox = 1.301

开态电流（在V_T）和关态电流（在V_GS = 0）之间的比率：
```
I_off = I_on × e^(-qV_T/(nkT))
```

计算：
```
qV_T/(nkT) = 1.11 / (1.301 × 0.0259) = 32.93
I_off = 5.556 μA × e^(-32.93)
I_off ≈ 2.4×10^-20 A
```

这是极小的。更实际地，考虑DIBL和其他短沟道效应：
```
I_off ≈ 几nA到100 nA（18nm技术的典型值）
```

## 总结

1. V_T = 1.11 V
2. S = 78 mV/decade
3. I_off ≈ 2.4×10^-20 A（理想），或几nA（考虑短沟道效应）

注：对于如此薄的氧化层（1.1 nm）和短沟道（18 nm），量子效应、隧穿和短沟道效应显著影响实际器件行为。
