# 作业2 - 问题4 (中文)

## 问题陈述
对于室温下的Si p+n长二极管，两侧均匀掺杂。已知N_D = 10^16 cm^-3，τ_p = 10^-8 s，D_p = 10 cm^2/s，V_A = 23kT/q。确定：
1. n-Si中的少子分布，即Δp_n(x)~x
2. 多子感应电流等于少子感应电流的x坐标，即J_n = J_p

## 解答

### 已知数据
- T = 300 K (室温)
- N_D = 10^16 cm^-3
- τ_p = 10^-8 s (少子寿命)
- D_p = 10 cm^2/s (空穴扩散系数)
- V_A = 23kT/q = 23 × 0.0259 = 0.596 V
- 对于Si：n_i = 1.5 × 10^10 cm^-3

### 第1部分：少子分布 Δp_n(x)

对于正向偏置下的长二极管，过剩少子载流子浓度遵循：

```
Δp_n(x) = p_n(x) - p_n0 = p_n0(e^(qV_A/kT) - 1) × e^(-x/L_p)
```

其中：
- p_n0 = n_i^2/N_D = (1.5×10^10)^2/10^16 = 2.25×10^4 cm^-3
- L_p = sqrt(D_p × τ_p) = sqrt(10 × 10^-8) = sqrt(10^-7) = 3.16 × 10^-4 cm = 3.16 μm
- e^(qV_A/kT) = e^23 ≈ 9.74 × 10^9

因此：
```
Δp_n(x) = 2.25×10^4 × (9.74×10^9 - 1) × e^(-x/3.16×10^-4)
Δp_n(x) ≈ 2.19×10^14 × e^(-x/3.16×10^-4) cm^-3
```

或更准确地：
```
Δp_n(x) = 2.19×10^14 × exp(-x/L_p)  其中 L_p = 3.16 μm
```

### 第2部分：J_n = J_p的位置

少子载流子电流（空穴电流）为：
```
J_p(x) = -qD_p × dΔp_n/dx = qD_p × Δp_n(0)/L_p × e^(-x/L_p)
```

在x = 0处：
```
J_p(0) = qD_p × Δp_n(0)/L_p
      = 1.6×10^-19 × 10 × 2.19×10^14 / 3.16×10^-4
      = 1.11×10^4 A/cm^2
```

准中性n区中的多子载流子电流（电子电流）：
```
J_n(x) = J_总 - J_p(x)
```

由于总电流恒定：
```
J_总 = J_p(0) = 1.11×10^4 A/cm^2
```

当J_n = J_p时：
```
J_n(x) = J_p(x)
J_总/2 = J_p(0) × e^(-x/L_p)
1/2 = e^(-x/L_p)
-x/L_p = ln(0.5) = -0.693
x = 0.693 × L_p = 0.693 × 3.16 μm
x ≈ 2.19 μm
```

## 总结
1. Δp_n(x) = 2.19×10^14 × exp(-x/3.16μm) cm^-3
2. J_n = J_p 在距离结约 x ≈ 2.19 μm 处
