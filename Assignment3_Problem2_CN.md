# 作业3 - 问题2 (中文)

## 问题陈述
给定一个NPN BJT，其中I_En = 100 μA，I_Ep = 1 μA，I_Cn = 99 μA，I_Cp = 0.1 μA，计算：
1. α_T
2. γ
3. I_E, I_C, I_B
4. α_DC和β_DC
5. I_CB0和I_CE0
6. 如果I_Cn增加到接近100 μA的值，而所有其他电流分量保持固定。I_Cn的增加对β_DC有什么影响？解释你的答案。
7. 如果I_Ep增加，而所有其他电流分量保持固定。I_Ep的增加对β_DC有什么影响？解释你的答案。

## 解答

### 已知数据
- I_En = 100 μA (从发射区注入的电子电流)
- I_Ep = 1 μA (注入发射区的空穴电流)
- I_Cn = 99 μA (到达集电极的电子电流)
- I_Cp = 0.1 μA (集电区的空穴电流)

### 第1部分：基区传输系数 α_T

α_T是从发射区注入的电子到达集电极的比例：
```
α_T = I_Cn / I_En = 99 / 100 = 0.99
```

### 第2部分：发射极效率 γ

γ是发射极电流中电子电流的比例：
```
γ = I_En / (I_En + I_Ep) = 100 / (100 + 1) = 100/101 ≈ 0.9901
```

### 第3部分：总电流 I_E, I_C, I_B

**发射极电流：**
```
I_E = I_En + I_Ep = 100 + 1 = 101 μA
```

**集电极电流：**
```
I_C = I_Cn + I_Cp = 99 + 0.1 = 99.1 μA
```

**基极电流（从电流守恒）：**
```
I_B = I_E - I_C = 101 - 99.1 = 1.9 μA
```

### 第4部分：直流电流增益 α_DC和β_DC

**α_DC（共基极电流增益）：**
```
α_DC = I_C / I_E = 99.1 / 101 ≈ 0.9812
```

或：
```
α_DC = γ × α_T = 0.9901 × 0.99 = 0.9802 ≈ 0.98
```

**β_DC（共发射极电流增益）：**
```
β_DC = I_C / I_B = 99.1 / 1.9 ≈ 52.2
```

或：
```
β_DC = α_DC / (1 - α_DC) = 0.9812 / 0.0188 ≈ 52.2
```

### 第5部分：反向饱和电流 I_CB0和I_CE0

**I_CB0** 是集电极-基极反向饱和电流：
```
I_CB0 = I_Cp (近似) = 0.1 μA
```

**I_CE0** 是集电极-发射极反向饱和电流：
```
I_CE0 = I_CB0 / (1 - α_DC) = 0.1 / (1 - 0.9812)
I_CE0 = 0.1 / 0.0188 ≈ 5.3 μA
```

或：
```
I_CE0 = (1 + β_DC) × I_CB0 = (1 + 52.2) × 0.1 = 5.32 μA
```

### 第6部分：增加I_Cn的影响

如果I_Cn增加到接近100 μA：

**新α_T：**
```
α_T,new = 100 / 100 = 1.0
```

**新α_DC：**
```
α_DC,new = γ × α_T,new = 0.9901 × 1.0 = 0.9901
```

**新β_DC：**
```
β_DC,new = α_DC,new / (1 - α_DC,new) = 0.9901 / 0.0099 = 100
```

**影响：** β_DC **增加** 从52.2到100。

**解释：** 增加I_Cn改善了基区传输系数α_T（基区中复合的电子更少），从而增加α_DC。由于β_DC = α_DC/(1-α_DC)，当α_DC接近1时，即使α_DC的小幅增加也会导致β_DC大幅增加。

### 第7部分：增加I_Ep的影响

如果I_Ep增加而其他分量保持固定：

**新γ：**
```
γ_new = I_En / (I_En + I_Ep,new) = 100 / (100 + I_Ep,new)
```

这会降低γ（发射极效率降低）。

**新α_DC：**
```
α_DC,new = γ_new × α_T
```

这会降低α_DC。

**新β_DC：**
由于α_DC降低，β_DC **降低**。

**解释：** 增加I_Ep意味着更多空穴从基区注入到发射区，降低了发射极效率γ。这降低了α_DC，进而降低了β_DC。晶体管效率降低，因为更多发射极电流"浪费"在空穴注入而非有用的电子注入上。

## 总结

1. α_T = 0.99
2. γ = 0.9901
3. I_E = 101 μA，I_C = 99.1 μA，I_B = 1.9 μA
4. α_DC = 0.98，β_DC = 52.2
5. I_CB0 = 0.1 μA，I_CE0 = 5.3 μA
6. 增加I_Cn会**增加** β_DC（更好的基区传输）
7. 增加I_Ep会**降低** β_DC（更差的发射极效率）
