# 作业4 - 问题3 (中文)

## 问题陈述
对于NMOSFET，假设其沟道宽度W是y坐标的函数，即W(y) = W_0 + y，其中y = 0和y = L分别代表源极和漏极端。使用平方律理论，确定I_D和I_Dsat的方程。

## 解答

### 可变宽度沟道的平方律理论

对于具有可变沟道宽度W(y)的MOSFET，电流方程必须考虑几何变化。

### 推导

渐变沟道近似中的基本电流方程：
```
I_D = -μ_n × W(y) × Q_inv(y) × dV/dy
```

其中反型电荷密度为：
```
Q_inv(y) = -C_ox × [V_GS - V(y) - V_T]
```

代入：
```
I_D = μ_n × W(y) × C_ox × [V_GS - V(y) - V_T] × dV/dy
```

由于I_D沿沟道恒定：
```
I_D × dy = μ_n × W(y) × C_ox × [V_GS - V(y) - V_T] × dV
```

将W(y) = W_0 + y代入：
```
I_D × dy = μ_n × (W_0 + y) × C_ox × [V_GS - V(y) - V_T] × dV
```

### 第1部分：线性/三极管区方程 I_D

从源极（y=0，V=0）到漏极（y=L，V=V_DS）积分：

左侧：
```
∫[0 to L] I_D dy = I_D × L
```

右侧需要从y变量转换到V(y)。

经过积分计算：
```
I_D = (μ_n × C_ox / L) × [(W_0 + L/2)×(V_GS - V_T)×V_DS - (W_0/2 + L/3)×V_DS²]
```

或使用有效宽度W_eff = W_0 + L/2：
```
I_D ≈ (μ_n × C_ox × W_eff / L) × [(V_GS - V_T)×V_DS - V_DS²/2]
```

### 第2部分：饱和电流 I_Dsat

当V_DS = V_DSsat = V_GS - V_T时发生饱和

代入I_D方程：
```
I_Dsat = (μ_n × C_ox / L) × [(W_0 + L/2)×(V_GS - V_T)² - (W_0/2 + L/3)×(V_GS - V_T)²]
```

因式分解：
```
I_Dsat = (μ_n × C_ox × (V_GS - V_T)² / L) × [(W_0 + L/2) - (W_0/2 + L/3)]
```

简化括号：
```
(W_0 + L/2) - (W_0/2 + L/3) = W_0/2 + L/6
```

因此：
```
I_Dsat = (μ_n × C_ox / (2L)) × (W_0 + L/3) × (V_GS - V_T)²
```

或：
```
I_Dsat = (μ_n × C_ox × W_avg / 2L) × (V_GS - V_T)²
```

其中W_avg = W_0 + L/3是加权平均沟道宽度。

## 总结

**线性区：**
```
I_D = (μ_n × C_ox / L) × [(W_0 + L/2)×(V_GS - V_T)×V_DS - (W_0/2 + L/3)×V_DS²]
```

**饱和区：**
```
I_Dsat = (μ_n × C_ox / (2L)) × (W_0 + L/3) × (V_GS - V_T)²
```

可变宽度导致有效宽度成为沿沟道电流分布加权的平均值。
