# 作业2 - 问题3 (中文)

## 问题陈述
一个在室温热平衡条件下的Si pn突变结，掺杂为N_A = 10^16 cm^-3和N_D = 10^15 cm^-3。确定：
1. V_bi
2. x_p, x_n 和 W_dep
3. E_max
4. 绘制电荷密度、电场和电势随坐标变化的图

## 解答

### 已知数据
- T = 300 K (室温)
- N_A = 10^16 cm^-3
- N_D = 10^15 cm^-3
- 对于300K的Si：n_i = 1.5 × 10^10 cm^-3, ε_s = 11.7ε_0 = 1.04 × 10^-12 F/cm

### 第1部分：内建电势 V_bi

```
V_bi = (kT/q) * ln(N_A * N_D / n_i^2)
V_bi = 0.0259 * ln(10^16 × 10^15 / (1.5 × 10^10)^2)
V_bi = 0.0259 * ln(10^31 / 2.25 × 10^20)
V_bi = 0.0259 * ln(4.44 × 10^10)
V_bi = 0.0259 * 24.52
V_bi ≈ 0.635 V
```

### 第2部分：耗尽区宽度

使用电荷中性条件：N_A * x_p = N_D * x_n

总耗尽宽度：
```
W_dep = sqrt(2ε_s/q × V_bi × (N_A + N_D)/(N_A × N_D))
W_dep = sqrt(2 × 1.04×10^-12 / 1.6×10^-19 × 0.635 × 1.1×10^16/10^31)
W_dep = sqrt(1.3×10^7 × 0.635 × 1.1×10^-15)
W_dep = sqrt(9.07×10^-9)
W_dep ≈ 9.52 × 10^-5 cm = 0.952 μm
```

根据电荷中性：
```
x_n = W_dep × N_A/(N_A + N_D) = 0.952 × 10^16/(1.1×10^16) = 0.866 μm
x_p = W_dep × N_D/(N_A + N_D) = 0.952 × 10^15/(1.1×10^16) = 0.0866 μm
```

### 第3部分：最大电场

```
E_max = -qN_D*x_n/ε_s = -qN_A*x_p/ε_s
E_max = 1.6×10^-19 × 10^15 × 0.866×10^-4 / 1.04×10^-12
E_max = 1.33 × 10^4 V/cm
```

### 第4部分：图示

**电荷密度 ρ(x):**
```
         |
   qN_D  |________________
         |                |
    0    |________________|________________
         |        -x_p    0    x_n
  -qN_A  |                |
         |________________|
         
ρ(x) = -qN_A  对于 -x_p < x < 0
ρ(x) = qN_D   对于 0 < x < x_n
ρ(x) = 0      其他区域
```

**电场 E(x):**
```
    E    |
    0    |     /\
         |    /  \
         |   /    \
  E_max  |  /______\______
         | -x_p    0    x_n
         
E(x) 从0线性增加到E_max (p侧)
E(x) 从E_max线性减少到0 (n侧)
在结处(x=0)达到峰值
```

**电势 V(x):**
```
    V    |           _____
  V_bi   |          /
         |         /
         |        /
    0    |_______/________
         |-x_p   0    x_n
         
V(x) 在耗尽区呈抛物线型
总电势降 = V_bi
```

## 总结
- V_bi = 0.635 V
- x_p = 0.087 μm
- x_n = 0.865 μm  
- W_dep = 0.952 μm
- E_max = 1.33 × 10^4 V/cm
