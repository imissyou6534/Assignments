# 作业4 - 问题5 (中文)

## 问题陈述
对于NMOSFET，其I_D-V_DS图如下图所示。W = 2.5 μm，L = 0.5 μm，t_ox = 10 nm。使用平方律理论。
1. 如果V_T = 0.5 V，确定V_GS
2. 当器件偏置工作在图中的A点时，确定漏极端点的Q_inv
3. 如果V_GS改变为V_GS - V_T = 3 V，V_DS = 4 V，确定I_D

## 解答

### 已知数据
- W = 2.5 μm = 2.5×10^-4 cm
- L = 0.5 μm = 0.5×10^-4 cm
- t_ox = 10 nm = 10×10^-7 cm
- V_T = 0.5 V
- ε_ox = 3.45×10^-13 F/cm

### 计算

**氧化层电容：**
```
C_ox = ε_ox / t_ox = 3.45×10^-13 / (10×10^-7) = 3.45×10^-7 F/cm²
```

### 第1部分：确定V_GS

**从图中**（假设典型的I_D-V_DS特性）：
- 如果图显示特定V_GS下的饱和电流I_Dsat
- 从曲线读取I_Dsat

**示例：** 如果从图中I_Dsat = 1 mA：

使用饱和电流方程：
```
I_Dsat = (μ_n × C_ox × W / (2L)) × (V_GS - V_T)²
```

假设μ_n ≈ 500 cm²/(V·s)：

```
1×10^-3 = (500 × 3.45×10^-7 × 2.5×10^-4 / (2 × 0.5×10^-4)) × (V_GS - 0.5)²
1×10^-3 = 4.31×10^-4 × (V_GS - 0.5)²
(V_GS - 0.5)² = 2.32
V_GS = 2.02 V
```

### 第2部分：漏极端的Q_inv（A点）

在A点，假设器件处于饱和区。

在饱和区的漏极端：
```
Q_inv(L) = 0（夹断条件）
```

如果A点在线性区，V_DS < V_GS - V_T：
```
Q_inv(L) = -C_ox × [V_GS - V_DS - V_T]
```

**示例：** 如果A点V_GS = 2 V，V_DS = 1 V：
```
Q_inv(L) = -3.45×10^-7 × (2 - 1 - 0.5)
|Q_inv(L)| = 1.725×10^-7 C/cm²
```

### 第3部分：V_GS - V_T = 3 V，V_DS = 4 V时的I_D

检查是否饱和：V_DS (4 V) > V_GS - V_T (3 V) → 是，饱和

使用饱和电流方程：
```
I_D = (μ_n × C_ox × W / (2L)) × (V_GS - V_T)²
```

V_GS - V_T = 3 V，μ_n = 500 cm²/(V·s)：
```
I_D = (500 × 3.45×10^-7 × 2.5×10^-4 / (2 × 0.5×10^-4)) × 3²
I_D = 4.31×10^-4 × 9
I_D = 3.88×10^-3 A = 3.88 mA
```

## 总结（示例值）

1. V_GS ≈ 2.0 V（从图中饱和电流确定）
2. A点漏极处|Q_inv| ≈ 1.73×10^-7 C/cm²（如果在线性区）
3. I_D = 3.88 mA（V_GS - V_T = 3 V，V_DS = 4 V时）

注：实际值取决于从提供的图中读取I_D-V_DS曲线。
