# 作业2 - 问题5 (中文)

## 问题陈述
对于室温下的Si pn突变结，载流子浓度的分布如图所示（未提供，但可以从典型问题推断）。
1. 该结是正向还是反向偏置？解释原因。
2. 准中性区中是否存在低电平注入？解释原因。
3. 确定N_A和N_D。
4. 确定施加的偏压V_A。

## 解答

### 假设
由于未提供图表，我将提供解决此类问题的一般方法。

### 第1部分：偏置方向

要确定结是正向还是反向偏置，需检查：
- **正向偏置**：结边界处的少子载流子浓度高于平衡态（Δn_p > 0, Δp_n > 0）
- **反向偏置**：结边界处的少子载流子浓度低于平衡态（接近零）

**如果 Δp_n(x_n) > 0 且 Δn_p(-x_p) > 0**：结为**正向偏置**
**如果 p_n(x_n) << p_n0 或 n_p(-x_p) << n_p0**：结为**反向偏置**

### 第2部分：低电平注入

低电平注入意味着：
- Δn_p << p_p0 (在p区)
- Δp_n << n_n0 (在n区)

其中：
- p_p0 ≈ N_A (p区多子载流子)
- n_n0 ≈ N_D (n区多子载流子)

**检查**：如果 Δp_n << N_D 且 Δn_p << N_A，则低电平注入成立。

通常，对于 V_A < 0.6V 的正向偏置，保持低电平注入。

### 第3部分：确定N_A和N_D

从载流子浓度图中：
- **N_D** ≈ n区多数电子浓度（平台水平）
- **N_A** ≈ p区多数空穴浓度（平台水平）

平衡态下的少子载流子浓度：
- p_n0 = n_i^2/N_D
- n_p0 = n_i^2/N_A

### 第4部分：确定施加的偏压V_A

从结边界处的少子载流子浓度：

对于n区中的空穴：
```
p_n(x_n) = p_n0 × e^(qV_A/kT)
```

因此：
```
V_A = (kT/q) × ln(p_n(x_n)/p_n0)
V_A = (kT/q) × ln((p_n0 + Δp_n(x_n))/p_n0)
```

同样，对于p区中的电子：
```
V_A = (kT/q) × ln(n_p(-x_p)/n_p0)
```

## 示例解答
假设从图中得到典型值：
- N_D = 10^16 cm^-3 (从n区平台)
- N_A = 10^17 cm^-3 (从p区平台)
- p_n0 = (1.5×10^10)^2/10^16 = 2.25×10^4 cm^-3
- p_n(x_n) = 10^10 cm^-3 (从图中结边界处)

则：
```
V_A = 0.0259 × ln(10^10/2.25×10^4)
V_A = 0.0259 × ln(4.44×10^5)
V_A = 0.0259 × 13.0
V_A ≈ 0.337 V (正向偏置)
```

低电平注入检查：
```
Δp_n = 10^10 - 2.25×10^4 ≈ 10^10 cm^-3
N_D = 10^16 cm^-3
Δp_n << N_D ✓ (低电平注入成立)
```

## 总结
解决方法取决于准确读取载流子浓度图以提取N_A、N_D和结边界处的少子载流子浓度。
