# 作业4 - 问题4 (中文)

## 问题陈述
对于NMOSFET，W/L = 5，t_ox = 20 nm，μ_n = 600 cm²/(V·s)，使用平方律理论。
1. 当器件工作在深度线性区（V_DS非常小但非零）时，导通电阻为500 Ω，确定Q_inv
2. 确定使导通电阻为500 Ω所需的V_GS - V_T

## 解答

### 已知数据
- W/L = 5
- t_ox = 20 nm = 20×10^-7 cm
- μ_n = 600 cm²/(V·s)
- R_on = 500 Ω
- ε_ox = 3.45×10^-13 F/cm

### 第1部分：确定Q_inv

在深度线性区（V_DS → 0），MOSFET表现为电阻。

导通电阻为：
```
R_on = 1 / (μ_n × (W/L) × |Q_inv|)
```

求解|Q_inv|：
```
|Q_inv| = 1 / (μ_n × (W/L) × R_on)
|Q_inv| = 1 / (600 × 5 × 500)
|Q_inv| = 1 / (1.5×10^6)
|Q_inv| = 6.67×10^-7 C/cm²
```

### 第2部分：确定V_GS - V_T

反型电荷与栅电压的关系为：
```
Q_inv = -C_ox × (V_GS - V_T)
```

因此：
```
V_GS - V_T = |Q_inv| / C_ox
```

计算C_ox：
```
C_ox = ε_ox / t_ox
C_ox = 3.45×10^-13 / (20×10^-7)
C_ox = 1.725×10^-7 F/cm²
```

现在：
```
V_GS - V_T = 6.67×10^-7 / 1.725×10^-7
V_GS - V_T = 3.87 V ≈ 3.9 V
```

### 验证

使用V_DS → 0的线性区方程：
```
g_ds = ∂I_D/∂V_DS = μ_n × C_ox × (W/L) × (V_GS - V_T)
```

导通电阻：
```
R_on = 1/g_ds = 1 / [μ_n × C_ox × (W/L) × (V_GS - V_T)]
R_on = 1 / [600 × 1.725×10^-7 × 5 × 3.87]
R_on = 500 Ω ✓
```

## 总结

1. |Q_inv| = 6.67×10^-7 C/cm² = 0.667 μC/cm²
2. V_GS - V_T = 3.9 V
