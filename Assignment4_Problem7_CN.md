# 作业4 - 问题7 (中文)

## 问题陈述
对于NMOSFET，W = 50 μm，L = 5 μm，t_ox = 50 nm，N_A = 3×10^15 cm^-3，μ_n = 800 cm²/(V·s)，使用重掺杂n+多晶硅作为栅电极，Q_ox = 0。
确定：
1. V_T
2. 当V_GS = 2 V时的I_Dsat
3. 当V_GS = 2 V和V_DS = 0 V时的g_ds
4. 当V_GS = 2 V和V_DS = 2 V时的g_m

## 解答

### 已知数据
- W = 50 μm = 50×10^-4 cm
- L = 5 μm = 5×10^-4 cm
- t_ox = 50 nm = 50×10^-7 cm
- N_A = 3×10^15 cm^-3
- μ_n = 800 cm²/(V·s)
- n+多晶硅栅，Q_ox = 0
- T = 300 K，n_i = 1.5×10^10 cm^-3
- ε_ox = 3.45×10^-13 F/cm
- ε_s = 1.04×10^-12 F/cm

### 初步计算

**氧化层电容：**
```
C_ox = ε_ox / t_ox = 3.45×10^-13 / (50×10^-7) = 6.9×10^-8 F/cm²
```

**费米电势：**
```
φ_F = (kT/q) × ln(N_A/n_i)
φ_F = 0.0259 × ln(3×10^15 / 1.5×10^10)
φ_F = 0.0259 × 12.21
φ_F = 0.316 V
```

### 第1部分：阈值电压V_T

对于p衬底上的n+多晶硅栅，V_FB ≈ 0（Q_ox = 0的理想情况）。

```
V_T = 2φ_F + (1/C_ox) × sqrt(4qε_s × N_A × φ_F)
```

计算耗尽电荷项：
```
Q_dep,max = -sqrt(4qε_s × N_A × φ_F)
Q_dep,max = -sqrt(4 × 1.6×10^-19 × 1.04×10^-12 × 3×10^15 × 0.316)
Q_dep,max = -2.51×10^-8 C/cm²
```

因此：
```
V_T = 2 × 0.316 + 2.51×10^-8 / 6.9×10^-8
V_T = 0.632 + 0.364
V_T = 0.996 V ≈ 1.0 V
```

### 第2部分：V_GS = 2 V时的I_Dsat

检查：V_GS - V_T = 2 - 1 = 1 V

饱和电流：
```
I_Dsat = (μ_n × C_ox × W / (2L)) × (V_GS - V_T)²
I_Dsat = (800 × 6.9×10^-8 × 50×10^-4 / (2 × 5×10^-4)) × 1²
I_Dsat = 2.76×10^-4 A = 0.276 mA
```

### 第3部分：V_GS = 2 V，V_DS = 0 V时的g_ds

在V_DS = 0 V时，器件处于线性区（深度三极管）。

漏-源电导为：
```
g_ds = ∂I_D/∂V_DS |V_DS=0 = μ_n × C_ox × (W/L) × (V_GS - V_T)
```

```
g_ds = 800 × 6.9×10^-8 × (50×10^-4 / 5×10^-4) × (2 - 1)
g_ds = 800 × 6.9×10^-8 × 10 × 1
g_ds = 5.52×10^-4 S = 0.552 mS
```

### 第4部分：V_GS = 2 V，V_DS = 2 V时的g_m

检查工作区：V_GS - V_T = 1 V，V_DS = 2 V
由于V_DS (2 V) > V_GS - V_T (1 V)，器件处于饱和区。

饱和区的跨导：
```
g_m = ∂I_Dsat/∂V_GS = μ_n × C_ox × (W/L) × (V_GS - V_T)
```

```
g_m = 800 × 6.9×10^-8 × (50×10^-4 / 5×10^-4) × 1
g_m = 5.52×10^-4 S = 0.552 mS
```

## 总结

1. V_T = 1.0 V
2. I_Dsat = 0.276 mA（V_GS = 2 V时）
3. g_ds = 0.552 mS（V_GS = 2 V，V_DS = 0 V时）
4. g_m = 0.552 mS（V_GS = 2 V，V_DS = 2 V，饱和区）
