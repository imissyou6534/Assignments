# 作业2 - 问题1 (中文)

## 问题陈述
使用耗尽区假设来分析和讨论线性渐变结。对于掺杂在耗尽区呈线性分布的pn结 N_D - N_A = αx，施加偏压V_A。确定以下公式：
1. 电场
2. 电势
3. 内建电势
4. 耗尽区宽度

## 解答

### 1. 电场

对于线性渐变结，其中 N_D - N_A = αx，α为梯度常数。

使用泊松方程：
```
dE/dx = -ρ/ε_s = -q(N_D - N_A)/ε_s = -qαx/ε_s
```

从0到x积分：
```
E(x) = -qαx²/(2ε_s) + C
```

在x = 0处，根据线性渐变结的对称性，E(0) = 0，所以 C = 0。

实际上，考虑适当的边界条件，设x_p（p侧延伸）和x_n（n侧延伸）。

电荷密度为：
```
ρ(x) = qαx  (对于 -x_p < x < x_n)
```

使用泊松方程和适当的边界条件：
```
E(x) = -qα/(2ε_s) * (x² - x_n²)  对于 0 < x < x_n
E(x) = -qα/(2ε_s) * (x² - x_p²)  对于 -x_p < x < 0
```

在x = 0（结点）处，最大电场为：
```
E_max = qα/(2ε_s) * (x_n² + x_p²)/2
```

根据电荷中性：对于线性渐变结 x_p = x_n，因此：
```
E_max = qαx_n²/ε_s
```

### 2. 电势

对电场积分：
```
V(x) = -∫E(x)dx
```

对于n侧 (0 < x < x_n)：
```
V(x) = qα/(2ε_s) * [x³/3 - x_n²x] + C
```

对于p侧 (-x_p < x < 0)：
```
V(x) = qα/(2ε_s) * [x³/3 + x_p²x] + C'
```

### 3. 内建电势

内建电势V_bi是结两侧的总电势降：
```
V_bi = V(x_n) - V(-x_p)
```

对于线性渐变结，x_p = x_n = W/2：
```
V_bi = qα/(12ε_s) * W³
```

或用x_n表示：
```
V_bi = qα/(3ε_s) * x_n³
```

### 4. 耗尽区宽度

施加偏压V_A时，总电压为V_bi - V_A。

根据关系式：
```
V_bi - V_A = qα/(3ε_s) * W³
```

求解W：
```
W = [3ε_s(V_bi - V_A)/(qα)]^(1/3)
```

这是施加偏压下线性渐变结的总耗尽宽度。

对于热平衡状态（V_A = 0）：
```
W₀ = [3ε_sV_bi/(qα)]^(1/3)
```

## 总结

线性渐变结与突变结的区别在于：
- 耗尽宽度随(V_bi - V_A)^(1/3)变化，而不是(V_bi - V_A)^(1/2)
- 电场分布是抛物线型而不是线性
- 内建电势与耗尽宽度呈三次方关系
