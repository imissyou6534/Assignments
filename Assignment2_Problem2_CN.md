# 作业2 - 问题2 (中文)

## 问题陈述
一个在室温下维持的Si pn突变结掺杂如下：p-Si侧 E_F = E_V - 2kT，n-Si侧 E_F = E_C - E_G/4。
1. 绘制该pn结的平衡能带图
2. 确定内建电势V_bi并在能带图中标记

## 解答

### 第1部分：能带图

对于p-Si侧：
- E_F = E_V - 2kT
- 这意味着费米能级在价带边缘下方2kT
- 空穴浓度可从以下公式得出：p = N_V * exp(-(E_F - E_V)/kT) = N_V * exp(2)

对于n-Si侧：
- E_F = E_C - E_G/4
- 这意味着费米能级在导带边缘下方E_G/4
- 电子浓度可从以下公式得出：n = N_C * exp(-(E_C - E_F)/kT) = N_C * exp(-E_G/(4kT))

在室温下（T = 300K）：
- kT ≈ 0.0259 eV
- 对于Si：E_G ≈ 1.12 eV

能带图（文字表示）：
```
p侧                  结                    n侧
                         
E_C  ----                                    ---- E_C
         \                                  /
          \                                /
           \                              /
            \                            /
  E_V - 2kT  \_ _ _ _ E_F _ _ _ _ _ _ _/  E_C - E_G/4
              \                          /
               \                        /
E_V  -----------\______________________/----------- E_V

         |<---- qV_bi ---->|
```

### 第2部分：内建电势计算

内建电势由平衡时的费米能级对齐决定。

在p侧，E_F到E_i（本征费米能级）的距离：
```
E_F - E_i = -E_G/2 + 2kT (在p侧)
φ_p = (E_i - E_F)/q = E_G/(2q) - 2kT/q
```

在n侧，E_i到E_F的距离：
```
E_F - E_i = E_C - E_G/4 - (E_C - E_G/2) = E_G/4 (在n侧)
φ_n = (E_F - E_i)/q = E_G/(4q)
```

内建电势为：
```
V_bi = φ_p + φ_n = E_G/(2q) - 2kT/q + E_G/(4q)
V_bi = 3E_G/(4q) - 2kT/q
```

在室温（T = 300K）下代入数值：
```
V_bi = 3(1.12)/(4) - 2(0.0259)
V_bi = 0.84 - 0.0518
V_bi ≈ 0.788 V ≈ 0.79 V
```

另一种计算方法：
从能带边缘的费米能级位置：
- 在p侧：E_V,p - E_F = 2kT
- 在n侧：E_F - E_C,n = -E_G/4

在平衡时，整个器件中E_F是恒定的。
内建电势是能带边缘位置的差：
```
qV_bi = (E_C,n - E_F) - (E_F - E_V,p)
     = E_G/4 - (-2kT)
     = E_G/4 + 2kT
V_bi = E_G/(4q) + 2kT/q
V_bi = 1.12/4 + 2(0.0259) = 0.28 + 0.0518 = 0.332 V
```

## 答案
内建电势 V_bi ≈ 0.33 V
